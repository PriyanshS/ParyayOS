# 3D Campus Models

Place your GLB/GLTF campus models here to view them in the Digital Twin:

- **campus1.glb** – Campus 1 (default)
- **campus2.glb** – Campus 2
- **campus3.glb** – Campus 3

If a file is missing, the app shows **DefaultComplex** (box buildings) instead.

You can export 3D models from Blender, SketchUp, or other tools as `.glb` and drop them here.
